# VS_pkvsconf

Simple VS Code extension to improve Explorer and project navigation.

## Features

- Reveal active file in Finder on macOS
- Root folder size in the status bar
- Project icon webview in Explorer
- Open GitHub repository from Source Control
- Extension tags
- Active page preview
- Workspace title bar color helper
- Exposed secrets detection
- Commit blocking when staged files contain secrets
- Agent Skills button to create the `.skills` symlink to the central `-agent` folder

## Commands

- Reveal Active File in Finder
- Refresh Root Folder Size
- Open GitHub Repository
- Catégorie (Add Tag)
- Rechercher des extensions
- Regenerer la couleur de la title bar
- Preview de la page en cours
- Afficher les secrets exposes
- Rescanner les secrets
- Commit (verification secrets)
- Agent Skills
